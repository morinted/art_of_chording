# ent-, int-

_Special thanks to Thomas Pacheco for funding this lesson._

This lesson covers the prefix compound that begins words like "enter" and "interact".

--------------------

We use just a single cluster for both "ent-" and "int-" briefs. The combination is arbitrary, `SPW`:

{% stenodisplay labels="all" %}
SPW
{% endstenodisplay %}

* `SPWER`: enter
* `SPWAOEUR`: entire
* `SPWAEUL`: entail
* `SPWAOEUS`: entice
* `SPWEL`: intel

A little more complex:

* `SPWABGT`: intact
* `SPWEPBD`: intend
* `SPWEPBGS`: intention
* `SPWERPB`: intern
* `SPWERPBL`: internal
* `SPWERPBT`: internet

We can add in sounds on the left hand, too:

* `SPWRUPT`: interupt
* `SPWRABGS`: interaction
* `SPWREU`: entry
* `SPWRAPBS`: entrance
* `SPWHREBGT`: intellect
